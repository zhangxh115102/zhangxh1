  • 考勤是如何计算的？

    回答：
    1、缺勤
    需进行补课且不计当次课酬
    2、迟到
    A、与学生协商后，学生同意延长课时，则延后课程并按照迟到1分钟1元的标准进行课酬扣除发放
    B、如学生不同意，则15分钟以内迟到1分钟2元并延后或另行补课，15分钟以上算严重迟到，扣除本课时30%的课酬并延后或另行补课，30分钟以上扣除本课时课酬的50%作为惩罚，同时需延后或另行补课。
    3、早退
    A、与学生协商后，学生同意早退，则按照早退1分钟1元的标准进行课酬扣除发放并另行补课
    B、如学生不同意，则15分钟以内早退1分钟2元并另行补课，15分钟以上算严重早退，扣除本课时30%的课酬并另行补课，30分钟以上扣除本课时课酬的50%作为惩罚，同时另行补课。
